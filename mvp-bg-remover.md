# Image Background Remover — MVP 需求文档

## 1. 项目概述

| 字段 | 内容 |
|------|------|
| 产品名称 | BG Remover（暂定） |
| 一句话描述 | 上传图片，自动移除背景，一键下载透明 PNG |
| 核心价值 | 3秒完成，无需注册，打开即用 |
| 目标用户 | 设计师、电商卖家、自媒体创作者 |

---

## 2. 用户故事

**主路径：**
> 用户打开网页 → 拖拽/选择图片 → 系统处理 → 显示结果 → 下载透明图

**边界情况：**
- 图片太大 → 前端压缩后再发
- 网络失败 → 提示重试
- Remove.bg 返回错误 → 提示用户

---

## 3. 功能需求

### 必须有（MVP）

| # | 功能 | 描述 |
|---|------|------|
| F1 | 图片上传 | 支持拖拽 + 点击上传，格式支持 JPG/PNG/WebP |
| F2 | 图片预览 | 上传后显示原图预览 |
| F3 | 背景移除 | 调用 Remove.bg API 移除背景 |
| F4 | 结果展示 | 显示处理后的透明图预览（棋盘格背景） |
| F5 | 一键下载 | 下载为 PNG 文件 |
| F6 | 加载状态 | 处理中显示进度/动画 |
| F7 | 错误提示 | 失败时友好提示 |

### 暂时不做

- 用户注册/登录
- 批量处理
- 历史记录
- 付费订阅
- API key 管理后台

---

## 4. 非功能需求

| 指标 | 要求 |
|------|------|
| 处理时间 | < 5秒（不含网络） |
| 支持图片大小 | ≤ 10MB |
| 兼容性 | 现代浏览器（Chrome/Safari/Firefox/Edge） |
| 移动端 | 必须可用（响应式） |
| 可用性 | 部署在 Cloudflare，全球可访问 |

---

## 5. 技术架构

```
前端：Cloudflare Pages（静态HTML + Vanilla JS）
后端：Cloudflare Worker（Node.js / TypeScript）
AI：Remove.bg API（已有账户）
域名：（待定）
```

**数据流：**
```
用户浏览器
  ↓ 上传图片（FormData）
Cloudflare Worker
  ↓ 转发
Remove.bg API
  ↓ 返回PNG
Worker → 浏览器
  ↓
直接下载
```

**环境变量：**
- `REMOVE_BG_API_KEY`：Remove.bg API 密钥

---

## 6. 页面流程

```
[首页]
  ↓ 拖拽图片
[上传中] → 显示进度条
  ↓ 完成
[结果页] → 显示原图 + 透明图对比
  ↓
[下载按钮] → 浏览器下载 PNG
```

---

## 7. UI 设计方向

- **风格**：简约工具风，白底，无干扰
- **主色调**：蓝/紫色系（专业感）
- **棋盘格背景**：展示透明图效果
- **交互**：拖拽区域有视觉反馈（边框变色）

---
